#### Task 1\.1\.1

---

In computer jargon, what's the difference between a **statement** and a **comment**?


##### My Solution

*While a statement is a line of code for a basic program action, a comment does not
represent a command and does not affect the execution of a program.*




#### Task 1\.1\.2

---

What does it mean to say a program is **portable**?


##### My Solution

*A program is portable if it can be executed by more than one type of processor.*




#### Task 1\.1\.3

---

In common English, what does the word **compile** mean?


##### My Solution

*The word compile means that a so-called source code written in a high-level language
that can be easily understood by humans is translated into a low-level language that
can be understood by the processor; the result is called object code, or executable.*




#### Task 1\.1\.4

---

What is an **executable**? Why is that word used as a noun?


##### My Solution

*An executable is a computer program written in a low-level language that is understood
directly by the processor; the word is a noun because it is another term for object code.*
