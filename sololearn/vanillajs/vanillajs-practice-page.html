1. Working with Variables

1.1 Declaring a Variable

Use the keyword var : 

<script>
	var answer;
	document.write(answer);
</script>

Output:

1.2 Assignment

Assign the value 42 : 

<script>
	var answer;
	answer = 42;
	document.write(answer);
</script>

Output:

1.2.1 Change the Assignment

Replace int with string : 

<script>
	var answer;
	answer = 42;
	answer = "Answer to the Ultimate Question of Life.";
	document.write(answer);
</script>
